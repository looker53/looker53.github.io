安卓官方模拟器使用

2021-05-06 1 min read # 随笔

使用安卓自带的模拟器理由有几点：

  • 无广告
  • 系统更新方便，最新的安卓系统都支持
  • 命令行运行，非常快速

1、使用 android studio 创建虚拟机

手工创建 avd 非常麻烦，可以通过 android studio 更加方便。创建好的 avd 会保存在用户目录的 .android/avd 目录下。

这里查看 android studio 和模拟器的安装

2、启动模拟器

  • 把 sdk/tools 目录下的 emulator.exe 重命名，否则会提示不支持 x86 cpu 情况。
  • 设置 sdk/emulator 为 path 变量，这样就可以使用该目录下的 emulator 程序。
# 列出所有虚拟机
emulator -list-avds 
# 启动 adb
adb start-server
# 运行指定虚拟机
emulator -avd android-11-pixel
